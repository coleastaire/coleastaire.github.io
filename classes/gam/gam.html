Behavior Trees

Nodes

  • Decorators
  • Composites (Control Flow Nodes)
    • Selector - Left to right, returns after all children finish
    • Sequence - Left to right, returns when a child fails
    • Simple Parralel - executes tasks and it's tree at the same time.
  • Services
    • Update function within a composite.
  • Tasks (Leaf Nodes)
    • Event Recieve Execute
      • recieves the signal to execute the scripts
    • Finish Execute
      • send the signal back and returns a boolean on success
